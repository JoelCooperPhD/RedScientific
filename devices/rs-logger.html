RS Logger

Data acquisition software for human factors research

Overview

RS Logger is data acquisition and logging software purpose-built for human factors research. It provides a unified interface for controlling Red Scientific devices, capturing synchronized data streams, and managing experimental sessions.

Designed by researchers for researchers, RS Logger handles the technical complexity of multi-device synchronization so you can focus on your study design and data analysis.

Key Capabilities

Device Control

Configure and control Visual Occlusion Goggles and DRT systems from a single interface. Set timing parameters, stimulus patterns, and recording options.

Synchronized Recording

Capture data from multiple devices with precise time alignment. All events are timestamped to a common clock for accurate cross-device analysis.

Real-Time Display

Monitor incoming data during sessions. View response times, hit rates, and device status in real time to catch issues before they affect your data.

Event Marking

Insert timestamped markers during recording to annotate task phases, conditions, or notable events. Markers are embedded in the data stream for easy segmentation.

Features

Precision Timing

Multi-device synchronization with sub-millisecond precision and TTL trigger support for external system integration.

Session Management

Track participants, configure recording parameters, and add session notes and metadata for organized data collection.

Live Monitoring

Real-time data visualization with keyboard shortcuts for event marking during experimental sessions.

Flexible Export

Export to CSV and common formats compatible with Excel, R, Python, SPSS, and other analysis tools.

System Requirements

OS: Windows 10+
RAM: 4GB minimum
Storage: 100MB
Ports: USB for devices
