Logger

Synchronized multi-sensor data collection for automotive and human factors research.

Cross-Platform Data Acquisition

Logger is a modular data collection platform designed for researchers who need synchronized capture of video, audio, eye tracking, and behavioral task data. Runs on Windows, macOS, Linux, and Raspberry Pi. All modules record with frame-level synchronization (~30ms accuracy) and export timestamped CSV data for analysis.

7 Modules Cameras, Audio, Eye Tracker, DRT, VOG, GPS, Notes
Synchronized Frame-level timing across all data streams
Cross-Platform Windows, macOS, Linux, Raspberry Pi
