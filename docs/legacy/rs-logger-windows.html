RS Logger (Windows)

Legacy Windows application for DRT and VOG devices.

Legacy Software

RS Logger for Windows is our original data collection software. While still supported for existing users, we recommend the new Logger for Raspberry Pi for new projects, which offers more modules and better synchronization.

Overview

RS Logger is a Windows application for collecting data from Red Scientific DRT and VOG devices. It was our first-generation data collection platform and remains available for researchers who need Windows compatibility.

Supported Devices

  • DRT — Detection Response Task (USB)
  • wDRT — Wireless Detection Response Task (XBee)
  • VOG — Visual Occlusion Glasses (USB)
  • wVOG — Wireless Visual Occlusion Glasses (XBee)

System Requirements

  • Windows 10 or Windows 11
  • USB 2.0 port
  • .NET Framework 4.7.2 or later

RS Logger vs. Logger

Migrating to Logger

If you're considering migrating from RS Logger to the new Logger platform:

What's the Same

  • Same DRT and VOG hardware devices work with both platforms
  • Same experiment modes (Cycle, Peek, eBlindfold, Direct)
  • Same ISO 17488 configuration parameters
  • Similar CSV output format (with additional sync columns in Logger)

What's Different

  • Logger runs on Raspberry Pi instead of Windows
  • Logger adds video, audio, eye tracking, GPS, and notes modules
  • Logger provides better cross-module synchronization
  • Logger is more portable (can run on battery power)

Hardware Required for Logger

  • Raspberry Pi 5 (or Pi 4)
  • microSD card (32GB+) or USB SSD
  • Power supply or battery pack
  • Your existing DRT/VOG devices

See the Getting Started guide for full setup instructions.
